# 動かし方メモ

- セットアップ
```
cd food-mbti
npm install
```

- TypeScript ビルド
```
npm run build
```

- 開発中（監視モード）
```
npm run dev
# tsc -w でファイル変更を監視
```

- サーバー起動
```
npm start
```


- 動作確認
Chrome で以下を開く
http://localhost:3000
